# SCT_DA_2_Data_Cleaning

## Overview

This project focuses on data cleaning and preprocessing using the Global Superstore dataset. The objective was to prepare the dataset for analysis by checking data quality, converting data types, and removing unnecessary information.

## Tasks Performed

* Loaded the dataset using Pandas
* Explored dataset structure and data types
* Checked for missing values
* Checked for duplicate records
* Converted `Order.Date` and `Ship.Date` to datetime format
* Removed unnecessary columns
* Exported the cleaned dataset to a CSV file

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Files

* `SCT_DA_2_DataCleaning.ipynb` - Jupyter Notebook containing all preprocessing steps
* `superstore.csv` - Original dataset
* `Cleaned_Global_Superstore.csv` - Cleaned dataset

## Outcome

Successfully cleaned and transformed the dataset, making it ready for further analysis and visualization.
