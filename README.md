# Task 2 - Data Cleaning and Preprocessing

## Objective

The objective of this project is to clean and preprocess the Global Superstore dataset using Python and Pandas.

## Tasks Performed

* Loaded the dataset using Pandas.
* Explored the dataset structure and data types.
* Checked for missing values.
* Identified duplicate records.
* Converted date columns (`Order.Date` and `Ship.Date`) to datetime format.
* Removed unnecessary columns for better data quality.
* Exported the cleaned dataset to a new CSV file.

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Dataset

Global Superstore Dataset

## Outcome

A cleaned and well-structured dataset ready for data analysis and visualization.
